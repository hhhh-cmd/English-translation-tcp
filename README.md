# English-translation-tcp
使用tcp cs模型实现词典翻译  

本项目主要用于学习使用 ，了解tcp通信机制及sqlite数据库使用，

# 1、文件介绍

server：实现tcp服务端口

clien：t实现客户端

dict.txt：里面是英文及中文的对应关系

readme：介绍文件

# 2、功能展示

1、客户端界面，通过不同的数字对应不同的功能，首先需要用户注册用户名和用户密码，使用sqlite数据库进行保存，以便下次用户登录验证

3、查询界面，输入对应的单词，即可得到对应中文意思
